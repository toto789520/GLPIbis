GLPI
GLPI (Gestionnaire Libre de Parc Informatique) est un logiciel open source destiné à la gestion des services informatiques d’une grande entreprise. GLPI permet de gérer les tickets ainsi que les ressources informatiques
hébergement
Pour l’hébergement de GLPI, cela se fait sur les serveurs du professeur en U300, sous Linux Debian.
Codage
Pour GLPI, nous avons choisi de créer un site web en codant en HTML, CSS, Python, et en utilisant une base de données SQL. Cela nous permet de contrôler totalement le site dans les moindres détails. Cela nous offre également l’opportunité d’acquérir de nouvelles compétences en codage
Open Source
GLPI sera un logiciel open source, ce qui signifie que tout le monde pourra le modifier, par exemple pour ajouter de nouvelles fonctionnalités ou changer le style graphique.
