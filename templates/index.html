A quoi sa sert ?

Présentation en 3 points de ce que c'est un GLPI

GLPI

Un GLPI permet de faire des demandes d'assistance (ticket). Par exemple, si une prise RJ45 fonctionne mal, une demande peut être faite et quelqu'un peut la régler

  • Plusieurs catégories
  • Système de priorité

Gestion d'inventaire

Un GLPI permet de gérer un inventaire. Cela est très pratique pour savoir le nombre d'un certain type de matériel (ex : PC) et de voir aussi leur référence

  • Trier par catégorie
  • Savoir la référence du matériel

Suivi d'activités

Ce GLPI permet de voir les activités en temps réel sur un écran qui sera dans le couloirs U300 sur cet écran on pourra voir les tâches qu'il y a à faire

  • Assignation de tâches par util
  • Suivi de progression

Informations sur le projet

Des informations plus techniques ci-dessous

Serveur

Pour l’hébergement de GLPI, cela se fait sur les serveurs prof en U300, sous Linux.

Code

Pour GLPI, nous avons choisi de créer un site web en codant en HTML, CSS, Python, et en utilisant une base de données SQL. Cela nous permet de contrôler totalement le site dans les moindres détails.

communauté (élèves)

Le code est totalement modifiable, donc cela signifie que de nouvelles fonctionnalités peuvent être ajoutées ou modifiées. Cela permet à n'importe quel élève de modifier le code.
Les profs doivent approuver le code avant d'être ajouté.
